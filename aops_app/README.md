## A. Initialize

### 1. Flask API
```bat
cd flask-backend
python -m venv env
env\Scripts\activate
pip install -r requirements.txt

```

### 2. Vuetify
```bat
cd vuetify-frontend
pnpm install

```

-----

## B. How to run
Prepare 2 `cmd`

### 1. Python Flask (Back-end API)

Open `cmd`, then paste this:

```bat
flask-backend\env\Scripts\activate
cd flask-backend\package
flask --debug run

```

### 2. Web Application (Front-end)

Open `cmd`, then paste this:

```bat
cd vuetify-frontend
pnpm run dev

```
